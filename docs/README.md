# Caloria — GitHub Pages (Gizlilik Politikası)

## Play Console URL

Pages etkinleştirildikten sonra:

```
https://topcu53.github.io/Caloria/privacy.html
```

## GitHub Pages açma (bir kez)

1. https://github.com/topcu53/Caloria → **Settings**
2. Sol menü **Pages**
3. **Build and deployment** → Source: **Deploy from a branch**
4. Branch: **main** → Folder: **/docs**
5. **Save**

1–2 dakika sonra site yayında olur.

## Güncelleme

`docs/privacy.html` dosyasını düzenleyip GitHub'a push edin.
